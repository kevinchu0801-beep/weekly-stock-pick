## 四因子（总分 100）

### 因子 1：动量与趋势（30）
- 60 日相对宽基 ETF 超额收益：0-15
- 20 日量价配合度（涨幅×换手率分位）：0-10
- MA60 站上 +5 / 站下 -5

### 因子 2：资金流与机构（30）
- 20 日主力净流入 / 流通市值：0-12
- 北向（A）/ 南向（港）/ 机构持仓变化（美）：0-10
- 30 日评级上调净数：0-8

### 因子 3：催化剂日历（20）⚠ 关键
- 30 天内财报：超预期 +15 / 平淡 +5 / 无 0
- 60 天内产品/政策/审批/订单：0-10
- 行业 β 顺风（板块 20 日涨幅前 30%）：0-5
- **封顶 20，因子3<8 直接踢**

### 因子 4：估值与基本面（20）
- PE/PB 5 年分位：<30% +10 / 30-50% +5 / >70% -5
- 最新季度营收 YoY：>20% +5 / 10-20% +3 / 负 -3
- 30 日 EPS 一致预期上修：0-5

## 硬约束

```
排名前提（任一不满足直接踢）：
1. 因子3 ≥ 8 分
2. 总分 ≥ 50 分
3. 同板块最多 2 只
4. 流动性达标

价格/市值只记录在底稿和卡片中，用于风险提示、拥挤度判断和仓位讨论，不作为直接剔除条件。
```

## 卡片必含 5 段

1. 一句话生意 + 护城河（≤40字，小学生能懂）
2. 核心催化（30-60天）：2-3 条具体事件 + 日期
3. 共识 vs 反共识：市场认为 / 我们认为
4. 风险点：2 条，明确失效条件（如"跌破 X 元 → 趋势破坏"）
5. 二元结论：分类 + 仓位 + 时间窗 + 止损 + 成功如果__/失败如果__

## 已知局限

- 数据延迟：周日跑的是上周五收盘价，催化剂时效有 1-2 个交易日偏差
- 北向资金日度数据 2024-08 起不公开，因子2 用替代指标（融资融券+大宗）
- 港股流动性悬崖：恒指外中小盘流动性可能被高估，建议分批建仓
- 因子权重未回测，前 4 周必须人工复核
